\#\# \mbox{\hyperlink{namespaceUI}{UI}} Element class 

Virtual parent class for all \mbox{\hyperlink{namespaceUI}{UI}} Elements. Contains generic methods to Enable/\+Disable, Show/\+Hide and modify the draw order.
