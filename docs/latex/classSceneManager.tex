\doxysubsection{Detailed Description}
\#\# \mbox{\hyperlink{classSceneManager}{Scene\+Manager}} class 

A container for all {\ttfamily \mbox{\hyperlink{classScene}{Scene}}} objects. Allows for the display of {\bfseries{only}} {\bfseries{one}} {\ttfamily \mbox{\hyperlink{classScene}{Scene}}} object at a time. \begin{DoxyNote}{Note}
Once given a pointer to a {\ttfamily \mbox{\hyperlink{classScene}{Scene}}} object, this class becomes owner of said object and will liberate their memory on it\textquotesingle{}s own. 
\end{DoxyNote}
