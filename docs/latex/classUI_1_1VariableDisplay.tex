\subsubsection*{template$<$typename T$>$\newline
class UI\+::\+Variable\+Display$<$ T $>$}

\#\# Variable Display class 

Allows for having a label with a variable value in it. The value will automatically update when the {\ttfamily \mbox{\hyperlink{classUI_1_1VariableDisplay_ad3fdb4bea2825dae65af40d29e4ce1b7}{Update()}}} method is called. \begin{DoxyNote}{Note}
This class stores a constant pointer to the variable it displays. Ownership of the object is not passed to this class. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
This template class is compatible with all data types compatible with the {\ttfamily std\+::to\+\_\+string()} method of the STL. 
\end{DoxyWarning}
