\doxysubsection{Detailed Description}
This file contains global constants, utility functions, and external references for the game system. 

This file defines some globally accessible constants, utility functions, and external variables used throughout the game system.

It includes paths for resources such as textures, which are located in the {\ttfamily /resources/textures} directory. These paths are defined relative to the executable\textquotesingle{}s location, and they allow easy access to external assets like textures.

This file also contains a reference to the {\ttfamily \mbox{\hyperlink{classSceneManager}{Scene\+Manager}}} object, which is used for managing the game\textquotesingle{}s scenes, as well as constants for controlling the draw order of game objects.
