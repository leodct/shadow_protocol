\doxysubsection{Detailed Description}
\#\# \mbox{\hyperlink{classScene}{Scene}} class 

A container for all {\ttfamily \mbox{\hyperlink{classGameObject}{Game\+Object}}} and {\ttfamily \mbox{\hyperlink{classUIContainer}{UIContainer}}} objects that should be displayed in the same place. \begin{DoxyNote}{Note}
Once given a pointer to a {\ttfamily \mbox{\hyperlink{classGameObject}{Game\+Object}}} or {\ttfamily \mbox{\hyperlink{classUIContainer}{UIContainer}}}, this class becomes owner of said object and will liberate their memory on it\textquotesingle{}s own. 
\end{DoxyNote}

\begin{DoxyWarning}{Warning}
The {\ttfamily \mbox{\hyperlink{classScene}{Scene}}} object is owner of all pointers given to it to facilitate centralized memory cleanups. This means the user is not required to call the {\bfseries{delete}} function on all of the {\ttfamily \mbox{\hyperlink{classScene}{Scene}}} object\textquotesingle{}s elements as the parent object will take care of it upon it\textquotesingle{}s destruction. 
\end{DoxyWarning}
