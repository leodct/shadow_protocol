\doxysubsection{Detailed Description}
\hypertarget{structTransform2D_autotoc_md1}{}\doxysubsubsection{The Transform2\+D Struct}\label{structTransform2D_autotoc_md1}
The {\ttfamily \mbox{\hyperlink{structTransform2D}{Transform2D}}} struct stores the position, rotation, and scale of an object in 2D space.


\begin{DoxyItemize}
\item {\ttfamily position}\+: A {\ttfamily Vector2} representing the position of the object in 2D space.
\item {\ttfamily rotation}\+: A float value representing the rotation of the object in radians.
\item {\ttfamily scale}\+: A float value representing the scaling factor of the object, where {\ttfamily 1.\+0f} is the default (no scaling). 
\end{DoxyItemize}
