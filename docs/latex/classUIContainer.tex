\doxysubsection{Detailed Description}
\#\# \mbox{\hyperlink{classUIContainer}{UIContainer}} class 

Stores and manages multiple {\ttfamily \mbox{\hyperlink{classUIElement}{UIElement}}} objects and their memory. \begin{DoxyNote}{Note}
The {\ttfamily \mbox{\hyperlink{classUIContainer}{UIContainer}}} class automatically manages the reserved memory of all it\textquotesingle{}s {\ttfamily \mbox{\hyperlink{classUIElement}{UIElement}}} objects. When given a pointer to an object, said object\textquotesingle{}s ownership will be passed to the respective {\ttfamily \mbox{\hyperlink{classUIContainer}{UIContainer}}} for centralized memory management. 
\end{DoxyNote}
