Detailed Description

## UIContainer class

Stores and manages multiple UIElement objects and their memory.

Note
The UIContainer class automatically manages the reserved memory of all it's UIElement objects. When given a pointer to an object, said object's ownership will be passed to the respective UIContainer for centralized memory management.
