Detailed Description

## Image Display class

Allows for displaying an Image/Texture2D as a UIElement.

Constructor & Destructor Documentation

◆ ImageDisplay()

UI::ImageDisplay::ImageDisplay ( Texture2D  texture,
Transform2D  _transform = {},
Vector2  _origin = {} 
)

Create an ImageDisplay object with default parameters.

Parameters
textureImage to be displayed.
_transformNew object's transform. Determines position, scale and rotation.
_originPoint of reference from which to draw the @p texture.
Note
Set @p _origin to {image.width / 2.0f, image.height / 2.0f} to have the image centered around @p _transform.position.
